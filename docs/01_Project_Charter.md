# Project Charter

Project: SpineCoach AI

Goal:
Build an adaptive AI bodyweight performance coach for adults with scoliosis.

Primary User:
Archie, 32 years old.

Training Philosophy:
- Bodyweight only
- Home friendly
- Progressive
- Long-term consistency
- Performance without unnecessary spinal loading

Primary Outcomes:
- Better posture control
- Improved spinal stability
- Increased strength
- Reduced pain
- Improved mobility
- Long-term progress tracking
